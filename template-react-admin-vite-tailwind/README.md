## Getting Started (dev mode)

- Cloning this repo
- `npm i`
- `npm run dev`

## Make production build

- `npm run build`
- `npm run serve`

## Requirement

- Node > v10
- NPM

## Under the hood

- Vite
- React
- Tailwind CSS
- PostCSS
- CSS Modules
- MobX State Tree
- React Router
